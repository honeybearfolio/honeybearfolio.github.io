## Dashboard

The main dashboard provides an at-a-glance view of your financial health:

- **Net Worth Chart** - Track your total net worth over time with customizable date ranges
- **Income vs Expenses** - See how much you're earning vs spending each month
- **Expense Breakdown** - View spending by category with an interactive pie chart
- **Asset Allocation** - Visualize how your investments are distributed

## Portfolio Tracking

For brokerage accounts, HoneyBear Folio provides powerful investment tracking:

- **Holdings View** - See all your positions with current value and gain/loss
- **Real-time Quotes** - Stock prices are fetched from Yahoo Finance
- **Historical Charts** - View price history for any holding
- **Cost Basis Tracking** - Automatically calculate cost basis across multiple purchases
- **Dividend Tracking** - Record and track dividend income

## Multi-Currency Support

HoneyBear Folio handles accounts and transactions in any currency:

- Create accounts in different currencies
- Automatic exchange rate fetching for currency conversion
- Historical exchange rates for accurate reporting
- All dashboards display values converted to your base currency

## Import & Export

### Import

Bring in data from banks, other apps, or spreadsheets:

- **CSV Import** - Most flexible, works with any bank export
- **JSON Import** - Structured data format
- **XLSX Import** - Excel spreadsheets

The import wizard includes:
- Column mapping preview
- Auto-detection of common formats
- Per-row progress reporting
- Automatic account creation when needed

### Export

Get your data out whenever you need it:

- **CSV Export** - Universal format for spreadsheets
- **XLSX Export** - Native Excel format with proper data types
- **Database Backup** - Full SQLite database file

## Auto-Fill Rules

Save time with automatic transaction categorization:

- Create rules based on payee name patterns
- Use exact match or regex patterns
- Auto-fill category, account, notes, and tags
- Preview rules before applying
- Prioritize rules for complex scenarios

Rules are applied:
- During manual transaction entry
- When importing data
- When editing existing transactions

## FIRE Calculator

Plan your path to Financial Independence, Retire Early:

- Set your FIRE target number
- Calculate based on your actual expenses
- See projected years to retirement
- Adjust assumptions for savings rate and returns
- Uses your real data for personalized projections

## Privacy Mode

Hide sensitive numbers when screen sharing or in public:

- Toggle privacy mode from the sidebar
- All amounts are masked while keeping currency symbols
- Charts show shapes without revealing values
- Toggle back instantly when needed
